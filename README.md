■概要
レビューサイトです。
認証機能、CRUD機能、検索機能を実装しております。

また、Docker、Bootstrapによるレスポンシブ対応、AWSによる公開もしています(ドメインは取っていません)

■機能
-ユーザ登録
-ログイン
-ゲストログイン
-検索
-パンくずリスト
-並び替え(評価順or新しい順)
-お気に入り登録・解除
-レビュー一覧
-レビュー投稿

■使用技術
①フロントエンド
　・PHP(Laravel_Blade)
　・JavaScript(Bootstrap,Carismaテンプレート)

②バックエンド
　・PHP(Laravel)

③データベース
　・MySQL

④インフラ
　・Docker
　・AWS(ECR,ECS,Fargate)
